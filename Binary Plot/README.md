# GeoChem Binary Plot

Geochemistry Data Analysis and Plotting.

## 即开即用交互式投图界面（Python）

该目录提供了一个基于 **Streamlit** 的地球化学 CSV 投图工具，支持：

- 上传 CSV（启动后默认不绘图，等待用户选择）。
- 横纵坐标支持两种来源：
  - 原始数值列
  - 对原始列进行四则运算后的结果（`+`、`-`、`*`、`/`，右操作数可选“列”或“常数”）
- **多列筛选**：可指定筛选列数量，并对每列选择保留取值，仅绘制满足全部条件的数据。
- **多维分类**：可指定分类列数量（最多 3 列）：
  - 第 1 列：颜色区分
  - 第 2 列：形状区分
  - 第 3 列：大小区分
- 在图上方显示当前绘制点数。
- 一键保存当前投图为 PNG / SVG / PDF。

## 使用方法

1. 安装依赖：

   ```bash
   pip install -r requirements.txt
   ```

2. 启动界面：

   ```bash
   streamlit run app.py
   ```

3. 浏览器打开后上传 CSV 并设置参数。

## 输入数据说明

- 推荐 CSV 第一行为表头。
- 作为横纵坐标的原始列应为数值列（例如 `SiO2`, `K2O`, `Na2O` 等）。
- 筛选列与分类列可使用任意文本或数值列（例如 `Type`, `Sub type`, `country`）。
